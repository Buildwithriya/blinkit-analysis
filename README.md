**# 🛒 Blinkit Sales \& Performance Analysis**



**## 📊 Business Case**



Blinkit is a leading quick-commerce platform where rapid, data-driven decision-making is critical. In this project, I have analyzed Blinkit's sales, inventory, and delivery data to extract actionable insights that drive operational efficiency and customer satisfaction.



**## 🏗️ Tech Stack Used**



\- \*\*Excel:\*\* Used for initial data profiling, handling raw datasets, and preliminary cleaning.

\- \*\*SQL Server (T-SQL):\*\* Performed data transformation by joining  relational tables to calculate key business metrics such as Total Revenue, Average Order Value (AOV), and ROAS.

\- \*\*Power BI:\*\* Developed an interactive executive dashboard using the advanced `.PBIP` (Power BI Project) format for better version control and data modeling.



\#**# 📂 Project Structure**



\- `EXCEL\_DATA/`: Contains processed and cleaned datasets in Excel/CSV formats.

\- `02\_SQL/`: Includes T-SQL scripts for schema creation and analytical queries.

\- `dashboard/`: Contains the Power BI Project infrastructure, including the Semantic Model (DAX) and Report definitions.



**## 💡 Key Business Insights**



\- \*\*Sales Trends:\*\* Analyzed monthly and yearly revenue growth patterns to identify peak demand periods.

\- \*\*Customer Sentiment:\*\* Evaluated the direct impact of product ratings on overall sales volume.

\- \*\*Delivery Logistics:\*\* Analyzed the correlation between delivery turnaround time and customer retention rates.



**## 🚀 How to Use This Repository**



1\. \*\*Database Setup:\*\* Execute the scripts in the `SQL/` folder within SQL Server Management Studio (SSMS) to recreate the analytical environment.

2\. \*\*Visualization:\*\* Open the `.pbip` file in the `dashboard/` folder using the latest version of Power BI Desktop to interact with the report.
